## 🤖 Cách học

Nói với AI instructor:
- **"Dạy module X lesson Y"** — Bắt đầu 1 lesson cụ thể
- **"Học tiếp"** — Tiếp lesson tiếp theo trong module hiện tại
- **"Giải thích [concept]"** — Deep-dive vào 1 khái niệm
- **"Quiz module X"** — Làm bài kiểm tra cuối module
- **"Xem tiến độ"** — Xem progress hiện tại
- **"Review bài tập"** — Nhờ review output trong practice/

## 📝 Nguyên tắc

1. **Mỗi lesson có Exercise** — Không chỉ đọc, phải làm
2. **2 case study song song** — ITO CRM (chính) + Logistics (phụ)
3. **Quiz dạng artifact** — Comment trả lời trực tiếp trên file
4. **Deliverable mỗi module** — Output thực tế trong `practice/`
5. **Tốc độ tự chọn** — Học bao nhiêu lesson/ngày tùy bạn
