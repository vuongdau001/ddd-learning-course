## Phần 3: Build vs Buy vs AI Decision

| Domain | Phân loại | Quyết định | Lý do (dùng 5-Question Framework) |

| | | | |

**5-Question Framework:**
1. Lợi thế cạnh tranh? → Build
2. Có SaaS phù hợp? → Buy
3. AI có thể tự động hóa? → AI
4. Cần customize nhiều? → Build
5. Thay đổi thường xuyên? → Build (own the change)
